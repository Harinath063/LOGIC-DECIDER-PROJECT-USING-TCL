# Digital Logic Simulator using Tcl/Tk

## 📌 Overview
This project is a GUI-based Digital Logic Simulator developed using Tcl/Tk. It allows users to input binary values and perform basic logic operations such as AND, OR, NOT, and XOR. The application visually displays results in real-time, making it useful for understanding digital logic concepts.

---

## 🎯 Features
- GUI-based application using Tcl/Tk
- Supports logic gates:
  - AND
  - OR
  - NOT
  - XOR
- Real-time output display
- Input validation (accepts only 0 or 1)
- Simple and user-friendly interface

---

## 🛠️ Technologies Used
- Tcl (Tool Command Language)
- Tk (GUI Toolkit)

---

## 🚀 How to Run

### Step 1: Install Tcl/Tk
Make sure Tcl/Tk is installed on your system.

### Step 2: Save the File
Save the program as:

logic_gui.tcl

### Step 3: Run the Program
Open terminal or command prompt and run:

wish logic_gui.tcl

---

## 🧠 How It Works

1. User enters binary inputs (0 or 1)
2. Selects a logic operation (AND, OR, NOT, XOR)
3. The program processes the inputs using logic expressions
4. Output is displayed instantly on the GUI

---

## 📊 Example

| Input A | Input B | Operation | Output |
|--------|--------|----------|--------|
| 1 | 1 | AND | 1 |
| 1 | 0 | OR | 1 |
| 1 | - | NOT | 0 |
| 1 | 0 | XOR | 1 |
